## 🚀 Flagship — Global Shopper

> **[globalshopper.in](https://www.globalshopper.in)** — a cross-border B2C e-commerce marketplace bringing **2M+ products to India** with all-inclusive INR pricing and 10–15 day delivery. Built and maintained **end to end, solo** — *not* Shopify, *not* a framework.

- **Backend (Node.js / Express):** a single service with **rate-limit-aware priority queues** and **3-layer caching** (in-memory → SQLite → disk) so a supplier API powers the whole store without ever tripping its limits.
- **Server-side pricing engine:** tamper-proof — FX, margin, per-unit freight quoting, deterministic synthetic discounts. Prices are never trusted from the client.
- **Payments:** Razorpay with **HMAC signature verification, authorize→capture, idempotency, and auto-refunds** on failure; a gated Cash-on-Delivery flow.
- **AI:** "**Maya**", an AI shopping assistant using **LLM tool-calling** that searches the live catalog & recommends complementary products — plus a **multilingual (Hindi/Hinglish) AI search parser**.
- **Marketing & analytics:** Meta **Pixel + server-side Conversions API** with browser↔server **event de-duplication**, a dynamic-ads product feed, GTM/GA4.
- **Email:** ZeptoMail SMTP with **SPF/DKIM** domain auth, fire-and-forget so it can never block checkout.
- **Auth & security:** email/password + **Google Sign-In (web + Android)**, server sessions, timing-safe compares, IDOR/SSRF hardening.
- **Performance & SEO:** SSR meta/OG tags, sitemaps, structured data; **stale-while-revalidate caching & deferred scripts** → sub-second cold loads.
- **Mobile:** companion **Android app** (Expo / React Native) shipped via EAS.

`Node.js` · `Express` · `Vanilla JS SPA` · `SQLite` · `Razorpay` · `Meta CAPI` · `ZeptoMail` · `LLM` · `Expo` · `Render`
